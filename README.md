# 🌿 Dandeli Ride – Resort Booking Website

A responsive travel and resort booking website for **Dandeli tourism**.
The website allows visitors to explore resorts, packages, rooms, and water activities available in Dandeli and easily contact for bookings.

---

## 🚀 Live Features

* 🏕️ Explore different **resorts in Dandeli**
* 🛏️ View **room options and pricing**
* 🌊 Discover **water activities** like rafting, kayaking, boating
* 📦 Check **package details**
* 📱 **Responsive design** for mobile and desktop
* 📞 Direct **Call & WhatsApp booking buttons**
* ✨ Smooth **ScrollReveal animations**
* 🎥 Resort **videos and image galleries**

---

## ⚙️ Installation

```bash

```

Open the project folder:

```bash
cd dandeli-ride
```

Run the project by opening:

```
index.html
```

in your browser.
